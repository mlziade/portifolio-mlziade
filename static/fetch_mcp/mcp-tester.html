MCP Fetch

A Model Context Protocol (MCP) server that enables LLMs to fetch and process web content. Based on the official Anthropic fetch implementation and deployed with both SSE and HTTP endpoints.

Browser Limitation This MCP server is designed for server-to-server communication and does not support direct browser access due to CORS restrictions. Please use one of the connection methods below instead.

Available Endpoints

SSE Endpoint

https://fetch-mcp.mlziade.com.br/sse
  • Transport: Server-Sent Events
  • Connection: Direct to Supergateway
  • Best for: Claude Desktop, VS Code, MCP Inspector

HTTP Endpoint

https://fetch-mcp.mlziade.com.br/mcp

How to Connect

For Claude Desktop:

Add this configuration to your Claude Desktop settings:

{
  "mcpServers": {
    "fetch": {
      "url": "https://fetch-mcp.mlziade.com.br/sse"
    }
  }
}

Command Line Testing (cURL):

Initialize the session:

curl -X POST https://fetch-mcp.mlziade.com.br/mcp \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "initialize",
    "params": {
      "protocolVersion": "2024-11-05",
      "capabilities": {},
      "clientInfo": {"name": "curl-test", "version": "1.0"}
    }
  }'

Fetch a URL:

curl -X POST https://fetch-mcp.mlziade.com.br/mcp \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "id": 2,
    "method": "tools/call",
    "params": {
      "name": "fetch",
      "arguments": {
        "url": "https://mlziade.com.br/about",
        "max_length": 5000
      }
    }
  }'
