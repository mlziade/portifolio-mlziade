Projetos

MovieStars

MovieStars é uma extensão do Chrome que ajuda os usuários a explorar avaliações de usuários e críticos, agregando detalhes de sites de entretenimento líderes. Ele fornece uma visão geral abrangente.

zllm

Zllm é uma implementação em Golang de uma API GenAI, executando modelos LLM locais através do Ollama. Inclui ferramentas como OCR local, modelos multimodais e tradução para aprimorar a utilização do LLM.

Dontpad List!

Dontpad List! é uma extensão do Chrome que permite aos usuários anexar texto destacado de páginas da web como uma entrada de lista em uma página Dontpad, com base no estilo de formatação de referência ABNT brasileiro.

Conways's Game of Life

Uma implementação em Python do Game of Life de Conway, simulando a evolução de um autômato celular com base em regras simples. Esta versão mostra os padrões e o comportamento dinâmicos do jogo.
