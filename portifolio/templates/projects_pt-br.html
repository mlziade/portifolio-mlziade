MovieStars

MovieStars é uma extensão para navegadores Chromium que ajuda os usuários a explorar avaliações de usuários e críticos, agregando detalhes de sites de avaliação. Ele fornece uma visão abrangente sobre filmes, séries e animes.

JS Extensão Chrome Web Scraping

Dontpad List!

Dontpad List! é uma extensão para navegadores Chromium que permite aos usuários anexar textos em highlight de páginas web como um item de lista em uma página Dontpad. Se baseia no estilo de formatação de referência ABNT brasileiro para websites.

JS Extensão Chrome Dontpad ABNT

zllm

Zllm é uma implementação em Golang/Fiber de uma API de IA Generativa, executando modelos LLM localmente através do Ollama. Inclui ferramentas como OCR local, modelos multimodais e tradução para auxiliar a utilização de LLMs.

Go Fiber API LLMs OCR Ollama Queue Jobs

ZLLM Chat

Uma interface de chat simples para interagir com a API ZLLM. Você pode perguntar sobre qualquer coisa, ou sobre o próprio projeto ZLLM!

Uma implementação em Python do Game of Life de Conway, simulando a evolução de células com base em regras simples. É chamado do primeiro jogo de zero players.
