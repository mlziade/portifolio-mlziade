MovieStars

MovieStars is a Chrome extension that helps users explore reviews from both users and critics, aggregating details from leading entertainment websites. It provides a comprehensive overview.

Zllm is a Golang implementation of a GenAI API, running local LLM models through Ollama. It includes tools like local OCR, multimodal models, and translation to enhance LLM utilization.

Dontpad List!

Dontpad List! is a Chrome extension that allows users to append highlighted text from webpages as a list entry to a Dontpad page, based on the Brazilian ABNT reference formatting style.
